# 🚀 React Native Learning Project

A modern React Native application built while learning and practicing React Native development concepts. This project demonstrates navigation, reusable components, state management, hooks, styling, and scalable project architecture.

## 📱 Features

- ⚛️ React Native & Expo
- 🎨 Modern UI Components
- 🧩 Reusable Components
- 📂 Organized Folder Structure
- 🪝 Custom Hooks
- 📦 State Management
- 🌙 Responsive Design
- 🔥 Fast Development Experience

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Naman762/Fullstack-real-state-app.git
```

### Navigate to the project

```bash
cd Fullstack-real-state-app
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```

or

```bash
npx expo start
```

## 📱 Run on Device

### Android

```bash
npm run android
```

### iOS

```bash
npm run ios
```

### Web

```bash
npm run web
```
